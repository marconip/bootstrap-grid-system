When to use class prefix: col, col-sm, col-md, col-lg, col-xl

Resize your browser screen to see resolution. If the resolution is...

equal to or greater then

1200px

use the class

col-xl

max container 1140px

equal to or greater then

992px

use the class

col-lg

max container 960px

equal to or greater then

768px

use the class

col-md

max container 720px

equal to or greater then

576px

use the class

col-sm

max container 540px

less than

576px

use the class

col

auto container
